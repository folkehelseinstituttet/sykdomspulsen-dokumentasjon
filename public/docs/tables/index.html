Unified variables

All accessible tables will contain these variables.

Variables Type Accepted values Description
granularity_time Text total, year, month, week, day, hour Temporal granularity
granularity_geo Text nation, county, municip, station Geographical granularity
location_code Text norge, countyXX, municipXX, stationXX The geographical location
border Integer 2019, 2020 The borders (kommunesammenslåing) that location_code represents
age Text e.g. 0, 1, 2, 0-4, 5-14, total Age in years
sex Text male, female, total Sex
year Integer YYYY Use function fhi::isoyear_n
week Integer 0, 1, 2, …, 53 Use function fhi::isoweek_n
yrwk Text YYYY-WW Use function fhi::isoyearweek
season Text e.g. 2014/2015, 2014/2014, 2014 Winter seasons are denoted by 2014/2015 (generally three options: weeks 30-29, weeks 40-20, or weeks 40-39), summer seasons are denoted by 2014/2014 (generally weeks 21-39), years are denoted by the year.
x Double 1, 2, 3, …, 23, 23.5, 24, …, 52 Use function fhi::x
date Date YYYY-MM-DD Always corresponds to the last date in the time period. E.g. if granularity_time=='week' then date is the Sunday of that week. fhidata::days is useful here.
